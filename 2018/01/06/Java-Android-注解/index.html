一 注解说明

注解(Annotation)，也叫元数据。从 JDK 1.5 开始，java 开始支持注解。它是JDK1.5及以后版本引入的一个特性，与类、接口、枚举是在同一个层次。它可以声明在包、类、字段、方法、局部变量、方法参数等的前面，用来对这些元素进行说明，注释。文档 JSR-269 将它规范化，在 JDK 1.6 写入编译器 javac 中。
